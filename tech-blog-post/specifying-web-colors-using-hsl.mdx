## HSL - Hue, Saturation, and Lightness

[Specifying colors for the web with HSL](https://caniuse.com/#feat=css3-colors) has good modern browser support (of course it doesn't work with IE 8 or earlier ðŸ™„)

### Hue

the color. It takes a value between `0deg` and `360deg`.

![](https://res.cloudinary.com/jastuccio/image/upload/v1555869262/blog/specifying-web-colors-using-hsl/hsl_color_wheel.png)

### Saturation

How much color. `0%` (gray) to `100%` (full saturation).

### Lightness

`0%`(black) to `100%`(white) 50% is the mid point and the most colorful.

example: `color: hsl(280deg, 0%, 50%);`

If you need transparency use HSLa `hsla(280deg, 0%, 50%, 0.8)`
